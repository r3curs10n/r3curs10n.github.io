This theme allows you to use a `<Playground>` MDX component that is powered by [Sandpack](https://sandpack.codesandbox.io/).

You can author it like so:

````md
<Playground template="react">

```js name=App.js active
import { NAME } from './constants.js'

export default function App() {
  return <h1>Hello {NAME}</h1>
}
```

```js name=constants.js readOnly
export const NAME = 'World'
```

</Playground>
````

The end result will be:

<Playground template="react">

```js name=App.js active

export default function App() {
  return <h1>Hello {NAME}</h1>

```


</Playground>
